# WebGIS_Earthquake_Visualization
interactive GIS platform using JavaScript, Leaflet, html and css to visualize earthquake data
This project utilizes Leaflet for interactive mapping, integrates USGS API for real-time earthquake data, and combines various technologies to create an immersive experience for users interested in earthquake patterns and trends.


Data Sources
Our earthquake data is sourced from the United States Geological Survey (USGS) Earthquake Hazards Program, providing reliable and up-to-date seismic information.
